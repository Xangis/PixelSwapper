PixelSwapper
An Image Manipulator
Version 1.23

PixelSwapper is a simple but powerful image editing application. It features the ability to resize, crop, mirror, rotate, and rescale images and allows conversion from one format to another. The interface allows a user to quickly and easily manipulate images via buttons or menus.

An image may be loaded from the File menu or dropped into the application window via drag-and-drop.

The status bar shows the currently loaded file, its width, height, and the scale at which it is displayed on the screen. PixelSwapper will automatically scale a loaded image to fit on the screen for ease of viewing and manipulation, but the resolution will remain unchanged unless the image is specifically rescaled or resized.

Menus:

File Menu

  • Load File - Loads a file from disk.
  • Save File - Saves the current file to disk, overwriting the previous version, if any.
  • Save File As - Asks what to name the current image, then saves it to disk.
  • Paste - Pastes image data from the clipboard. This is especially handy for acquiring screenshots. To take a screenshot, hit SHIFT + PRINT SCREEN and then select Paste. To capture an image of the currently selected window to the clipboard, hit ALT + PRINT SCREEN and then select Paste. Nothing will happen if there is no image data on the clipboard.
  • Revert - Reverts to the saved file. Undoes any changes. Note that it is impossible to use the revert function if the image was acquired from the clipboard. The file that will be reverted to is displayed on the status bar.
  • Page Setup - Lets you set page settings for printing.
  • Print - Sends the image to the printer.
  • Print Preview - Shows a preview of what the image will look like when printed.
Transform Menu
  • Crop - Removes any part of the image that is outside of the currently selected area. A selection can be made by clicking and dragging the mouse to draw a green rectangle over the area to be kept.
  • Mirror Horizontal - Reflects the image about a horizontal line through the center.
  • Mirror Vertical - Reflects the image about a vertical line through the center.
  • Rescale Larger - Doubles the image size.
  • Rescale Smaller - Cuts the image size in half.
  • Resize - Shows the resize dialog. The old size in pixels is shown and a new size is requested.
  • Rotate Left - Rotates the image 90 degrees counterclockwise.
  • Rotate Right - Rotates the image 90 degrees clockwise.
  • To Grayscale - Converts the image to grayscale, removing all color.
Help Menu
  • Help - Shows this document.
  • About - Shows program credits.

Buttons:

Crop - Removes any part of the image that is outside of the currently selected area. A selection can be made by clicking and dragging the mouse to draw a green rectangle over the area to be kept.
Mirror - Left-clicking mirrors the image vertically, while right-clicking mirrors the image horizontcally.
Rescale - Left-clicking rescales the image smaller, while right-clicking rescales the image larger.
Resize - Shows the resize dialog. The old size in pixels is shown and a new size is requested.
Revert - Reverts to the saved file. Undoes any changes. Note that it is impossible to use the revert function if the image was acquired from the clipboard.
Rotate - Left-clicking rotates the image 90 degrees clockwise, while right-clicking rotates the image 90 degrees counterclockwise.
